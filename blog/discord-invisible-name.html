Discord Invisible Name

January 18, 2024
7 min read

You can’t set a truly empty username, but you can make your server nickname look blank by using an invisible Unicode character. This guide shows the safest way to do it and what to try when Discord trims your “blank” name.

Copy Invisible Characters Common Questions

Best invisible character for Discord (start here)

The most reliable option is usually a character that renders as a blank “space” in common fonts, but is not a normal ASCII space. If one character fails, try another from the tool, because Discord validation can change.

Fast method

  1. Go to TheInvisibleCharacter.live and copy an invisible character.
  2. Open Discord and edit your Server Profile (nickname) or Profile.
  3. Paste the invisible character as your nickname and save.

Desktop: make your Discord nickname look blank

  1. In Discord, open the server where you want the blank nickname.
  2. Click your name in the member list or open Server Settings → Profiles.
  3. Find Server Nickname (or “Nickname”).
  4. Paste an invisible character, then click Save.

Mobile: make your Discord nickname look blank

  1. Open the server.
  2. Tap the server name → Settings → Change Nickname (or Profiles).
  3. Paste the invisible character and confirm.

Troubleshooting

  • Discord says “nickname is invalid”: try a different invisible character (some are blocked).
  • It looks like a small box: that font doesn’t render the character as blank. Try another one.
  • It gets trimmed: avoid putting the invisible character only at the very start or end; some trimming rules apply.
  • Server requires unique nicknames: you may need a mix of invisible + normal characters, or a different invisible character sequence.

Use responsibly

A blank nickname can be fun, but it can also confuse moderation and verification workflows on some servers. Use it respectfully and follow each server’s rules.
