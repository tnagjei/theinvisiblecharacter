Cleanup workflow

Detect and remove invisible characters

Use the homepage detector as the review step: paste text, reveal code points, remove unwanted hidden Unicode, then copy cleaned text. This keeps cleanup explicit instead of silently changing content.

Reveal hidden Unicode code points

Paste text into the detector to show code points such as U+200B, U+200C, U+200D, U+2060, U+FEFF, and U+00A0 before publishing, importing, or moderating it.

Remove invisible characters after you inspect them

The remover is a cleanup action after review. It should not be used as a blind filter on language text, emoji sequences, or fields where joiners may be intentional.

What the remover cleans

Common cleanup targets include zero-width marks, BOM, word joiner, directional marks, Braille blank, Hangul filler, and non-breaking spaces.

Safe cleanup boundaries

Unsafe: do not use hidden Unicode cleanup to bypass AI detection, remove AI watermarks, hide identity, or make moderation harder. This page is for formatting repair and review.

Platform notes

Different apps filter different blanks

Use the simplest character first. If the app removes it, switch character instead of repeating the same failed paste.

TikTok

Start with U+200B, then U+2800 or U+3164. Usernames are stricter than display names and bios.

Instagram

Invisible characters are usually more useful for bios, captions, and separators than for usernames.

Discord and games

Fonts and moderation rules can expose or remove blank-looking marks, so verify from another device.

Invisible symbols

Symbol, character, blank text, and hidden character usually describe the same Unicode tool family.
