# 2026-07-10 TheInvisibleCharacter Full Site UI

* 用途：展示 qiaomu-design 第二阶段的四个真实设计方向。
* 关键入口：`index.html`。
* 边界/依赖：生产页面尚未修改；`selection.json` 由本地预览服务在用户确认后生成。
> 一旦本目录内容变化，请更新本文件

## Files

* `index.html`：四方向样机、三拨盘和选择入口。
* `selection.json`：用户确认方向后的回传记录，由服务生成。

## Design Read

* 页面类型：SEO 工具站与 Unicode 工作台。
* 受众：希望立即复制、检测和清理不可见字符的普通用户。
* 默认拨盘：视觉冒险度 6、动效强度 4、信息密度 6。
* 功能契约：3 秒内看懂用途，首屏可复制常用字符，保留搜索、检测、隐私与平台导航。

## Direction Isolation

* A：明亮、瑞士工业、无衬线与等宽、7:5 工具索引。
* B：暗色、未来实验室、衬线与等宽、命令带与双面板。
* C：暖纸、编辑杂志、衬线与人文无衬线、标题与标本带。
* D：明亮多色、玩具感、展示体与圆润无衬线、大操作键与全宽检测栏。
* 发散种子：`继续全站重构` 共 6 个字符，`6 mod 14 = 6`，命中“编辑杂志”，落实为方向 C。

## Reference DNA

* Resend：方向 B 借用 1px 冰蓝结构线、等宽代码作为主视觉、无传统阴影的多面板工作台。
* Wise：方向 D 借用 9999px 功能胶囊、30px 主工具面板圆角、重字标题与边框成层，不照搬荧光绿。

## Render QA

* 第一轮发现：移动端缩放样机偏到右下角、缺失 favicon 导致 500、方向 C 中段空白过大。
* 第二轮修复：样机改为绝对居中缩放、增加内联 favicon、补回方向 C 编辑说明块。
* 已验证：桌面 2×2、375px 与 430px 单列、无水平溢出、每个样机完整居中、4 个按钮无重复。
