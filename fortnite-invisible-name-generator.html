How to test a blank Fortnite name

Test one character at a time, then verify inside the launcher and in the game UI before assuming it worked.

    Copy

  3. 3

    Preview

    Verify

    Open Fortnite and compare how the name appears in menus and match UI.

FAQ

Fortnite invisible name FAQ

Does this prove Fortnite accepts blank names?

No. It gives copy options and a test workflow. Epic validation is pending until you check it yourself.

Which character should I try first?

Start with U+3164, then test U+2800 or U+200B. Keep a visible fallback ready before saving.

What if the name is rejected?

Paste a visible backup name, save that, and avoid repeating the same failed blank character.

Can console players use this?

Console display names usually route through the linked account system. Test in the account settings flow for your platform.

Is a hidden name safe?

It can create moderation and readability issues. Use visible names when identity clarity matters.
