Pour commencer

Qu'est-ce qu'un caractère invisible ?

Un caractère invisible (ou texte invisible, espace vide) est un caractère Unicode qui ne s'affiche pas à l'œil nu, mais que l'ordinateur et les applications traitent comme un vrai caractère. Contrairement à la barre d'espace normale, souvent supprimée ou refusée par les plateformes, ces caractères spéciaux comptent comme du contenu : ils permettent d'envoyer un message apparemment vide ou de laisser un champ rempli sans rien écrire.

Espace normal vs caractère invisible

La touche Espace (U+0020) est souvent ignorée par les applis. Un caractère invisible comme U+3164 est traité comme une lettre : il passe la validation là où un espace échoue.

Pourquoi les plateformes les bloquent

WhatsApp, Discord ou les formulaires nettoient parfois ces caractères pour empêcher l'abus. C'est pourquoi chaque outil ci-dessous propose un caractère de secours.

Est-ce légal et sûr ?

Oui pour un usage personnel (pseudo amusant, message vide, mise en forme). Évitez de les utiliser pour tromper ou contourner les règles d'une plateforme.

Chaque carte indique le meilleur usage, le risque de plateforme et une option de secours.

Zero Width Space

U+200B

Usage: bio, légende, séparation discrète.

Risque: souvent supprimé en début ou fin de champ.

Secours: U+2800 ou U+3164.

Braille Pattern Blank

U+2800

Usage: blanc visible dans certains noms et messages.

Risque: peut devenir un carré selon la police.

Secours: U+200B ou U+3164.

Hangul Filler

U+3164

Usage: noms, champs de profil, tests de secours.

Risque: certaines plateformes le normalisent.

Secours: U+2800 pour un blanc large.

Word Joiner

U+2060

Usage: empêcher une coupure de ligne.

Risque: retiré par les nettoyeurs de texte.

Secours: U+200B pour une séparation.

Zero Width Non-Joiner

U+200C

Usage: empêcher la jonction de caractères.

Risque: change la forme du texte.

Secours: U+200B.

Zero Width Joiner

U+200D

Usage: jonction invisible, surtout pour emoji et scripts.

Risque: résultat visuel inattendu.

Secours: U+200C ou U+200B.

Invisible Separator

U+2063

Usage: séparation technique invisible.

Risque: filtré dans les formulaires.

Secours: U+200B.

Function Application

U+2061

Usage: notation mathématique.

Risque: mauvais choix pour les noms sociaux.

Secours: U+200B.

Invisible Times

U+2062

Usage: multiplication invisible dans les formules.

Risque: filtré hors contexte mathématique.

Secours: U+2061 ou U+200B.

Left-to-Right Mark

U+200E

Usage: stabiliser un texte de gauche à droite.

Risque: modifie le sens d'affichage.

Secours: U+200B.

Right-to-Left Mark

U+200F

Usage: stabiliser un texte de droite à gauche.

Risque: édition plus difficile.

Secours: U+200E ou U+200B.

Left-to-Right Embedding

U+202A

Usage: test avancé de texte bidirectionnel.

Risque: influence le texte qui suit.

Secours: U+200E.

FAQ

Questions fréquentes

Qu'est-ce qu'un caractère invisible ?

C'est un caractère Unicode qui ne s'affiche pas à l'écran mais compte comme du texte pour les applications. On l'appelle aussi texte invisible ou espace vide.

Le texte est-il envoyé à un serveur ?

Non. La copie et les tests de base fonctionnent dans votre navigateur, sur votre appareil.

Pourquoi un caractère invisible disparaît-il ?

L'application peut nettoyer les espaces invisibles ou normaliser Unicode. C'est pourquoi chaque outil propose un caractère de secours.

Quel caractère invisible essayer d'abord ?

Commencez par U+200B (Zero Width Space). Essayez U+2800 ou U+3164 si le champ le supprime.

Puis-je envoyer un message vide sur WhatsApp ?

Oui. Copiez U+2800 (Braille Pattern Blank) et collez-le dans le champ de message : il part comme un message apparent vide.

Est-ce gratuit et sans inscription ?

Oui. Le générateur de caractère invisible est entièrement gratuit, sans compte et sans installation.
