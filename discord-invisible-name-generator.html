Safety

Discord safety and moderation notes

Do not impersonate

Invisible names can make identity harder to read. Use them only for formatting or personal style where the server allows it.

Expect server filters

Reported: moderation bots may flag zero width text, blank nicknames, or repeated invisible marks.

Keep the detector nearby

Use the local detector to reveal U+200B, U+2800, U+3164, and other hidden Unicode before pasting into public chats.

FAQ

Discord invisible name FAQ

Which Discord blank character should I try first?

Start with U+3164, then U+2800, then U+200B. Mark the result as reported until you test it in your own client.

Can I make my Discord username blank?

Usually no. Unique usernames are stricter than server nicknames, so use this tool mainly for nicknames, display names, and spacing tests.

Why did my blank nickname turn visible?

The font, Discord client, or server filter may show a box or remove the character. Switch characters and test again.

Does this still work in 2026?

It is pending until you test your exact Discord client and server. This page provides copy options and a verification workflow.

Is invisible text safe for servers?

Normal formatting is low risk, but hidden names can confuse moderation and accessibility tools. Follow server rules.

How do I remove hidden Unicode?

Paste the text into the homepage detector, review the code points, and copy a cleaned version after removing the hidden marks.
